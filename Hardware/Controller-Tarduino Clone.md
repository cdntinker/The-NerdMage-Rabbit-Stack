# Connectors & Jumpers:

| Connector | Purpose                                         | Notes                         |
| --------- |:-----------------------------------------------:| ------------------------------|
| J1        | Programming connections (from Programmer Board) |                               |
| J2        | USB 2.0 pass through                            |                               |
| J3, J4    | PD/POE output voltage                           | Not actually used here        |
| J5, J6    | Secondary output voltage                        | See jumper JP1                |
| J7        | SPI Connection                                  |                               |
| J9        | I2C connection                                  |                               |
