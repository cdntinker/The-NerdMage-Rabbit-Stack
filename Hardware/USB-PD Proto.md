# Connectors & Jumpers:

| Connector | Purpose                                             | Notes                         |
| --------- |:---------------------------------------------------:| -----------------------------:|
| J2        | USB 2.0 pass through (from PD Board)                |                               |
| J3, J4    | PD output voltage (from PD Board)                   |                               |
| J5, J6    | Secondary output voltage (from PD Board)            |                               |
| J7        | Control/status connection to PD Board               |                               |
